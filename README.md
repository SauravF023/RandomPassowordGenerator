# 🔐 Random Password Generator

A modern and responsive password generator built using HTML, CSS, and JavaScript that helps users generate secure and random passwords instantly.

## 📌 Overview

Random Password Generator is a lightweight web application that creates strong and secure passwords with customizable options. Users can select password length, include uppercase letters, include special characters, check password strength, and copy generated passwords with a single click.

The application also keeps track of recently generated passwords through a password history section.

---

## 🚀 Features

### Password Generation

* Generate 4 unique passwords simultaneously
* Randomized password generation
* Adjustable password length (4–32 characters)

### Customization Options

* Include/Exclude uppercase letters
* Include/Exclude special characters
* Numeric characters included by default

### Password Strength Indicator

Passwords are automatically classified as:

* Weak
* Medium
* Strong

Strength is calculated based on:

* Password length
* Uppercase letters
* Numbers
* Special characters

### Copy Functionality

* Click any generated password to copy it instantly
* Visual confirmation after successful copy

### Password History

* Stores the last 5 generated passwords
* Copy passwords directly from history

### User Interface

* Clean modern design
* Responsive layout
* Dark theme UI
* Interactive hover effects

## ⚙️ How It Works

### Step 1: Configure Options

Choose:

* Password length
* Include uppercase letters
* Include special characters

### Step 2: Generate Passwords

Click:

```text
⚡ Generate Passwords
```

The application creates 4 random passwords based on selected settings.

### Step 3: Check Strength

Each password is automatically labeled:

```text
Weak
Medium
Strong
```

### Step 4: Copy Password

Click on any generated password box to copy it to your clipboard.

### Step 5: View History

The last 5 generated passwords are stored in the history section.

## 🔒 Password Generation Logic

Character sets used:

```javascript
Lowercase Letters:
abcdefghijklmnopqrstuvwxyz

Uppercase Letters:
ABCDEFGHIJKLMNOPQRSTUVWXYZ

Numbers:
0123456789

Symbols:
!@#$%^&*()_+[]{}|;:,.<>?
```

Passwords are generated by randomly selecting characters from the active character pool.

---

## 🧠 Password Strength Rules

| Condition                  | Score |
| -------------------------- | ----- |
| Length ≥ 8                 | +1    |
| Length ≥ 12                | +1    |
| Contains Uppercase         | +1    |
| Contains Number            | +1    |
| Contains Special Character | +1    |

### Classification

```text
0–2 Points → Weak
3–4 Points → Medium
5 Points → Strong
```

## 🐞 Known Limitations

* Password history is not saved after page refresh
* Password generation uses Math.random() instead of cryptographically secure APIs
* No guarantee that every selected character type appears in generated passwords
